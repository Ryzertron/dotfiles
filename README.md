## Installation

This repository is structured to suit the GNU Stow requirements. You are welcome to try out my config by cloning this repo and
using GNU Stow to mirror it to your home directory.

```bash
# Clone the repository
git clone https://github.com/Ryzertron/dotfiles.git ~/dotfiles

# Change directory
cd ~/dotfiles

#Use GNU Stow to mirror
stow .

# This repo is still under development. I will be adding a install script soon 
# so that the necessary application and dependencies will be installed
```
